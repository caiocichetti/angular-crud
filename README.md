 ## Instalação
 
 **Clone o repositorio**
 
 ``` 
 cd backend
 npm install
 npm start
 ```
 
**se estiver no sistema linux ou mac**
 ``` 
 cd front end
 sudo npm i -g @angular/cli
 npm start
 ```
 
 **se estiver no Windows**
 ``` 
 cd front end
 npm i -g @angular/cli
 npm start
 ```
 
 Após aparecer "Compiled sucessfully"
 ```
 ng serve
 ```
 
 Após isso é so abrir o seu navegador e colocar na barra de pesquisa localhost:4200
